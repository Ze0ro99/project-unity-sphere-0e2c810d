# PiRC-101 Security & Risk Mitigation

| Threat | Impact | Mitigation Strategy |
| :--- | :--- | :--- |
| **Wash Trading** | High | **Hybrid Decay Model**: Once Pi leaves a verified Snapshot wallet, it loses its $W_m$ (Mined) status permanently. |
| **Oracle Poisoning** | Critical | **Medianized Feeds**: Cross-referencing 3+ decentralized oracles to confirm the $0.2248$ base price. |
| **Liquidity Drain** | Medium | **Exit Throttling**: Progressive fees on large-scale internal-to-external conversions. |
