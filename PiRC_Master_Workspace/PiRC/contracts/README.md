# PiRC Smart Contract Architecture

This directory contains reference contract modules for the PiRC protocol.

These contracts represent a conceptual implementation of the PiRC economic coordination system.

Modules:

token/
Defines the protocol token logic.

treasury/
Manages protocol reserves and treasury allocation.

reward/
Implements reward distribution logic.

liquidity/
Controls liquidity incentives and trading interaction.

governance/
Defines governance mechanisms for adjusting protocol parameters.

bootstrap/
Handles initial protocol configuration.

These contracts serve as reference implementations for simulation and research.
