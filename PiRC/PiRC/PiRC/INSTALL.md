# PiRC Installation & Activation Guide

## Prerequisites

- **Termux / Linux:** `pkg install rust python nodejs git`
- **Rust Toolchain:** `rustup target add wasm32-unknown-unknown`
- **Soroban CLI:** `cargo install --locked soroban-cli`

## Quick Start (One-Click Activation)

Simply run the master activation script. It will automatically compile all 150+ smart contracts, compute topological geometry, generate quantum entanglement keys, and activate the 7 colored layers:

```bash
./activate_pirc_full.sh
```
