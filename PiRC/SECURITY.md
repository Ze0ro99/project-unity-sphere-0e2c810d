# Security Policy

## Supported Versions
Only the current major version is supported with security updates. We monitor the ecosystem using GitHub Advanced Security and Automated Dependabot tracking.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
To report a security issue, please contact `security@pirc.network`. **DO NOT** open a public issue.
The Sovereign Engineering Team will respond within 48 hours to assess and patch the vulnerability.
